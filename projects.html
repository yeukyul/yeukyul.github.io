College Explorer

A interactive webapp that visualizes the top 800 colleges ranked by Times Higher Education during year 2011-2016. The application aims to provide viewer an insight into the global trends of the top university, and also learning basic statistics of each and one of these colleges. The project is created using Shiny in R, plotly, leaflet, and HTML widgets. The dataset provided is from Kaggle.com.

Harry Potter

A web based game created in Javascript that intends to recreate classic settings in Harry Potter book 1: Harry Potter and the Philosopher's Stone. Uses 2.5D space visualization. The game is co-created with Haiyun Tang.

Foo Bar

Foo Bar is a self-initiated web development project that I am currently working on with another CS student. It is a customizable blog that allows users to upload articles and photos. The blog is powered by CSS, HTML, Javascript, JQuery, Djangos, and Git. The final phase will be done in end of November 2016, during Thanksgiving.

GGPlot: Package Extension in R

An R package extension of existing R graphing package GGPlot. The extension aims at adding additional graph type to the current package including time series plot, boxcox graphs, and linear diagnostics plot. The study will be conducted under the supervision of Statistical faculty member Sam Ventura in Spring 2017.

RAPT

A middle school algebra peer tutoring agent with social awareness, developed in Articulab, Carnegie Mellon University. My responsibility includes analysing levels of algebra problems given to children, and using Bayesian learning method to derive and program a model that would allow Rapt agent to determine appropriate questions to give to learners given their algebra background and learning history.

SARA

A virtual agent that possesses social awareness, developed in Articulab, Carnegie Mellon University. My responsibility includes creating user study system, conducting user studies, and analysing data collected thereafter. We as a user study team team are working to send a paper in the importance of social awareness in virtual agents into CHI, the biggest Human Computer Interaction conference in the world.

Kickstarter

A static data visualization project that summarizes the statistics of top 800 most backed projects on Kickstarter. The visualization gains insight into information like amount funded, description length, project genre, and number of backers every project and provide key findings in this dataset.

Simple Proxy

A simple multi-threaded, interactive proxy that listens and accepts incoming requests. It then forward all incoming HTTP request to corresponding server. The proxy is written in C; It is tested and is compatible with Mozilla FireFox Explorer.

Impact-a-thon

During spring 2016, I participated a competition organized by Carnegie Mellon the college of engineering, aiming to implement positive impact in the society to foster acceptance of disabled children. The prompt of the Spring 2016 competition was to create an accessible playground for children with disabilities. We worked in a group of 5, with groupmates from disciplines all across Statistics, Electrical Engineering, and Mechanical Engineering. Our solution was to create tracks around playground that would allow children with difficulties to walk to be on a cart and enjoy the sensations that the playground facilities would bring.

Social Constructionism Research

In Fall 2014, I have worked on a small scale Interpretative Phenomenological Analysis (IPA) qualitative research regarding anti-immigration sentiment in France. The mini-research involved analysing transcript and careful intepretation with understanding of social constructionism. The end project was selected and presented in Dietrich Undergraduate Colloqium, organized by the Dietrich College of Humanities and Social Sciences.
