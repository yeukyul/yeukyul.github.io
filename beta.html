Lindia: Linear Diagnostic Plot Graphing Package

Statistical Graphing Package | Developed in R

  • Graphing diagnostics plots for linear models is painful, and so I developed an R package "lindia" (stands for linear diagnostics) under supervision of CMU Statistics Professor Sam Ventura. This package allows Statisticians to diagnose how well a linear model fits a dataset in one line of code. The package is now in Beta testing phase.
  • Mechanism:The package is built based on a an exisiting R graphing package GGPlot, developed by Hadley Wickham.
  • View more on what it does and how to download: https://github.com/yeukyul/lindia

Twins Paradox

Adventure Game | Developed in Python (pygame)

  • Purpose:A 2.5D space adventure game implemented in pygame module in python.
  • Story Behind: The story behind Twin's Paradox is based on the famous Physics Twins Paradox. It states that if one of the twins went travel around blackhole, when he comes back, he would find his brother on Earth aged much faster than he does.
  • The game is centered around this idea, where player is trapped in outter space, trying to reach home before his brother dies.
